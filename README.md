# kitbuilder

A tool to create random drumkits based on sounds on the internet from platforms like FreeSound.

## Supported Platforms

- FreeSound
- SampleFocus

## Installation

- Install one of the .zip files from the releases page
    - Releases come bundled with empty configuration files
- Build from source using `build.sh`

## Running

1. Copy `config.example.json` -> `config.json`
2. Populate FreeSound credentials with your API credentials
3. Change OR remove the Google Chrome path option
4. Change the number of sounds you want, or the categories
5. Run the tool
